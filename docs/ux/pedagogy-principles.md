# Pedagogy Principles

## 1. Show, then name
Let the learner see the geometric change before overwhelming them with notation.

## 2. Keep one concept visually dominant
Each module should have a clear “main idea”.

Examples:
- Dot product → projection / alignment
- Cross product → area + normal direction
- Determinant → area/volume scaling
- Eigen → preserved directions

## 3. Synchronize multiple representations
When useful, keep these in sync:
- scene
- formula
- prose
- highlighted geometric objects

## 4. Use edge cases intentionally
Presets should include:
- perpendicular vectors
- parallel vectors
- negative scaling
- determinant zero
- orientation flip
- rotation with no real eigenvectors

## 5. Encourage prediction
Before revealing the result, ask the learner to guess:
- sign
- direction
- magnitude trend
- whether a quantity should vanish

## 6. Avoid false precision
If a visualization is approximate or sampled, do not present it like an exact symbolic truth.

## 7. Favor understanding over feature count
A smaller, clearer module is better than a broad but confusing one.
